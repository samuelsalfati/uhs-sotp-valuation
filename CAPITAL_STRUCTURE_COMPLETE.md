### Voting Rights Structure ‚≠ê CRITICAL

#### Director Election (Special Voting Structure)

**Two Separate Voting Groups:**

1. **Group 1: Class A + Class C** (10.8% of shares)
   - Elect **5 directors** (majority of board)
   - Control **90.5% of voting power** on director elections
   - Each share = 1 vote

2. **Group 2: Class B + Class D** (89.2% of shares)
   - Elect **2 directors** (minority of board)
   - Control only **9.5% of voting power** on director elections
   - Each share = 1 vote

#### General Matters (Non-Director Votes)

All shares vote together, but with **different vote weights:**

| Class | Votes Per Share | Condition |
|-------|----------------|-----------|
| **Class A** | 1 vote | Always |
| **Class B** | **0.1 vote** (one-tenth) | Always |
| **Class C** | **100 votes** | **IF** holder owns 10√ó Class A shares |
| **Class C** | 1 vote | If condition not met |
| **Class D** | **10 votes** | **IF** holder owns 10√ó Class B shares |
| **Class D** | 0.1 vote | If condition not met |

**Example:**
- 1,000 Class C shares + 10,000 Class A shares = 100,000 votes (100 votes per Class C share)
- 1,000 Class C shares + 5,000 Class A shares = 1,000 votes (1 vote per Class C share - failed condition)
